# Transcription Pipeline

## Overview
This project implements a simple transcription pipeline that converts audio files into text with timestamps.

## Features
- Accepts audio files (WAV, MP3, etc.)
- Converts audio to standard format
- Transcribes using Whisper
- Returns timestamps per segment
- Handles long audio via chunking
